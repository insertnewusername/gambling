<h1>ONEKEY RNG</h1>

Onekey RNG is a duck RNG game, where you press J to earn a random amount of coins, and the use those coins to roll a duck!
There are 6 different duck colours (as for now), for you to obtain. Each one is of a different rarity.

Spinning for a duck will cost you 100 coins.

Have fun with the game!
